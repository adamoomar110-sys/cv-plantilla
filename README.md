# Premium Portfolio Template | Glassmorphism & AI-Driven Design

Esta es una plantilla de CV / Portfolio de alto impacto diseñada para profesionales de tecnología. Utiliza una estética de cristal (Glassmorphism), animaciones fluidas y un diseño centrado en la experiencia del usuario.

## ✨ Características
*   **Diseño Premium:** Estética moderna con fondos animados y efectos de desenfoque.
*   **Cursor Interactivo:** Un puntero personalizado que reacciona a los elementos de la interfaz.
*   **Animaciones Sutiles:** Efectos de entrada al hacer scroll y tipografía dinámica.
*   **Totalmente Responsive:** Adaptado para móviles, tablets y escritorio.
*   **SEO Optimizado:** Estructura semántica lista para buscadores.

## 🛠 Cómo Usar
1.  Clona este repositorio o descarga los archivos.
2.  Abre `index.html` y reemplaza los textos entre corchetes `[TU NOMBRE]`, `[TU PROFESIÓN]`, etc., con tus propios datos.
3.  Coloca tu foto de perfil con el nombre `profile_photo.png` en la carpeta raíz (o cambia la ruta en el HTML).
4.  Ajusta los enlaces de redes sociales en el `footer`.

## 📜 Estructura de Archivos
*   `index.html`: Estructura principal y contenido.
*   `style.css`: Estilos visuales y sistema de diseño.
*   `script.js`: Interactividad y animaciones.
